# E-Commerce Management System

A full-stack E-Commerce Management System built using **React.js, Node.js, Express.js, PostgreSQL, and Sequelize ORM**.

This project includes:

* User Authentication
* Product Management
* Order Management
* Inventory Tracking
* Admin Dashboard
* Cart & Checkout System

---

# Tech Stack

## Frontend

# Authentication Flow

1. User registers/logs in
2. JWT token is generated
3. Token is stored in LocalStorage
4. Protected routes verify token
5. Authorized users access protected resources
